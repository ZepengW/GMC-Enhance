GMC-Enhance 设置

适用于快捷键 J / L（Alt+Shift+J / Alt+Shift+L）
适用于快捷键 U / I（Alt+Shift+U / Alt+Shift+I）；Alt+Shift+O 重置
适用于快捷键 < / >（Alt+Shift+< / Alt+Shift+>）

说明：本扩展已内置快捷键监听与跨标签控制，无需依赖 Chrome 全局快捷键设置。

默认快捷键

  • 切换控制视频：Alt+Shift+V（在各标签视频间循环选择）
  • 播放/暂停：Alt+Shift+K（作用于当前选中视频）
  • 快退/快进：Alt+Shift+J / Alt+Shift+L（步长见上方设置）
  • 减速/加速：Alt+Shift+U / Alt+Shift+I（步长见上方设置）
  • 重置倍速：Alt+Shift+O
  • 音量- / 音量+：Alt+Shift+< / Alt+Shift+>（步长见上方设置）
  • 静音/恢复：Alt+Shift+M
  • 截图：Alt+Shift+S
